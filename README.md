# Movie Catalog App (Full-stack)

Навчальний проєкт для каталогізації фільмів.

## 🚀 Технології
- **Backend:** NestJS, TypeORM, MySQL
- **Frontend:** React (Vite), TypeScript, Axios, React Hook Form, Zod
- **API:** RESTful API

## 🛠 Функціональність
- Перегляд списку фільмів
- Додавання нових фільмів з валідацією
- Видалення фільмів
- Обробка помилок та стани завантаження
